vim-prove-syntax
================

High vimsibility syntax for [prove](http://search.cpan.org/dist/Test-Harness/bin/prove) syntax.

This syntax highlight 4 important messages to make fast and unstressful reading.

* Any errors on test
* Failed test
* Succeeded test
* Test result
